Hi, Welcome To My Blog!

My name is Jack Snowdon, I am a budding Full Stack Web Developer based in Wolverhampton, UK. I have launched this blog platform, in which i'll be updating with my latest projects, what's been influencing me and a hot take or two!
This is a work in progress, as well as updating the blog, i'll be adding new features and UX changes, which i'll also blog about on this platform. As a guest user you can view any blog post, upon registration you're given the power to comment upon a blog post.

About Me!

I'm 26 year old musician turned coder, reciving a First Class Honours from Code Institute for my studies in Full Stack Web Development after years of working in bars, cinemas and phone sales. When i'm not behind the drums or laptop I enjoy gaming of all forms, be it PC Gaming, classic consoles or tabletop games.
I also recently lost 7 Stone (or 100lbs!) through lifestyle changes, which is more of a humble brag at this point! However, i've found myself capable of helping others with their weight or mental health, which is something i've taken great pride in being able to help!

How It's Made!

This blog is powered The Wonderful Django Python Framework, which has been an incredible tool during my development as a developer!
Under the blog platform is a tiered access system, which allows users with the correct permissions access to not only posting/edting blog posts but access to a personal to-do system and a few other tools i'm currently working on. My idea is to launch tools slowly into this site as my confidence and network grows.
